API Overview

The University Student Portal API provides comprehensive endpoints for managing student information, courses, grades, and administrative operations.

Base URL
http://localhost:8080/api
Authentication

HTTP Basic Authentication

Username: admin, Password: admin123
Response Format

JSON

API Version

v1.0.0

Student Management

Manage student records, enrollment, and academic information.

GET /api/students
Get all students
GET /api/students/{id}
Get student by ID
POST /api/students
Create new student
PUT /api/students/{id}
Update student
DELETE /api/students/{id}
Delete student
Course Management

Manage course catalog, enrollment, and course information.

GET /api/courses
Get all courses
GET /api/courses/{id}
Get course by ID
POST /api/courses
Create new course
POST /api/courses/{id}/enroll/{studentId}
Enroll student in course
DELETE /api/courses/{id}/unenroll/{studentId}
Unenroll student from course
Grade Management

Manage student grades, GPA calculations, and academic performance.

GET /api/grades
Get all grades
GET /api/grades/student/{id}
Get grades by student
GET /api/grades/student/{id}/gpa
Calculate student GPA
POST /api/grades
Create new grade
Dashboard & Analytics

Access dashboard statistics and system analytics.

GET /api/dashboard/stats
Get dashboard statistics
GET /api/dashboard/student/{id}
Get student overview

HTTP Status Codes
Success Codes
  • 200 OK - Request successful
  • 201 Created - Resource created
  • 204 No Content - Resource deleted
Error Codes
  • 400 Bad Request - Invalid input
  • 401 Unauthorized - Authentication required
  • 404 Not Found - Resource not found
  • 409 Conflict - Resource already exists
  • 500 Internal Server Error
